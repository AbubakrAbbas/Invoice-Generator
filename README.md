# 🧾 Invoice Generator

A clean, simple invoice generator built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just open and use.

## 🔗 Live Demo

[View Live Site](https://AbubakrAbbas.github.io/Invoice-Generator)

---

## ✨ Features

- Add and remove line items dynamically
- Auto-calculates row amounts and grand total in real time
- Custom bill preview modal — no browser print dialog
- Download the bill as an HTML file
- Clear the entire invoice with one click

---

## 🚀 How to Use

1. Fill in the **From** and **Bill To** fields with company/address details
2. Add your line items — enter a description, quantity, and rate
3. The amount and grand total update automatically as you type
4. Click **Preview Bill** to see the formatted invoice in a modal
5. Click **Download Bill** to save it as an HTML file you can open anytime
6. Click **Clear Invoice** to reset everything and start fresh
